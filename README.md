## Anonymous-Mode
> $ git clone https://github.com/yuu0910/anonymous-mode/

> $ cd ./anonymous-mode/

> $ chmod 0755 ./anonymous

## Help
> $ sudo anonymous

## Stop
> $ sudo anonymous stop

## Start
> $ sudo anonymous start debian-tor

## Restart
> $ sudo anonymous restart

## Anonymous-Mode DNS Only
> $ chmod 0755 ./firewall

> $ sudo firewall start debian-tor
