## Anonymous-Mode
> $ git clone https://github.com/yuu0910/anonymous-mode/

> $ cd ./anonymous-mode/

> $ chmod 0755 ./firewall

## Help
> $ sudo ./firewall

## Stop
> $ sudo ./firewall stop

## Start
> $ sudo ./firewall start debian-tor anon

## Restart
> $ sudo ./firewall restart

## Start Arch Linux (Beta)
> $ sudo ./firewall start tor anon
